# Nuxt.js starter project with GraphQL-API (with SSR) in TypeScript

Starter project containing:
- Nuxt.js with Vuetify.js and SCSS
- express REST-API with GraphQL-API (Apollo)
- serverless(offline) configuration
- docker(compose) configuration with nginx

## Install dependencies
```shell
$ yarn install
```

## Starter Scripts

**fe** for front-end, **be** for back-end

- **fe:dev**: launches Nuxt.js front-end in dev-mode (with hot-reloading)
- **fe:build**: launches Nuxt.js build-process
- **fe:generate**: launches Nuxt.js generate-process
- **fe:start**: launches Nuxt.js (launch after **fe:build**)
- **be:dev**: launches API-server with nodemon
- **be:start**: launches API-server
- **start:sls**: launches serverless offline
- **start:docker**: launches front- and back-end (use only within docker-container)
